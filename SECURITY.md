# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please:

1. **Do NOT** open a public issue
2. Email security reports to: [Create security advisory](https://github.com/N3urda/pypi-mono/security/advisories/new)
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond within 48 hours and keep you updated on the fix progress.

## Security Best Practices

- Dependencies are automatically scanned by Dependabot
- Code is linted with ruff before merging
- All PRs require review before merge
